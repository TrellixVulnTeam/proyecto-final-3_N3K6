Cursos que ofrecemos

Estos son los cursos que se realizan en esta institucion.

Curso de Desarrollo Web

Aqui integraras todos los primeros pasos hacia el mundo de la programacion, tiene como principal objetivo adquirir todo el conocimiento base sobre este mundo para llevarte hacia los demas escalones.

Curso de JavaScript

Este es el siguiente paso en tu camino a ser programador, si ya tienes conocimiento sobre el anterior curso no hay problema, puedes empezar de inmediato con este y seguir aprendiendo para convertirte en un profesional.

Curso de Backend

Creiste que el frontend te serviria para todo? Bueno pues creiste mal. Aqui aprenderas la otra cara de la programacion, este es el curso final, la ultima parte por la que estudiaste todas esas horas en las anteriores clases. Este es el curso final para desarrollarte como todo un profesional.

Nuestras Instalaciones

Aqui aprenderas todo sobre nuestras instalaciones.

Salas De Computacion

Salas de computacion, comunmente llamadas en nuestra institucion, aulas. Aqui pasaras la mayor parte del tiempo estudiando todo lo relacionado a este mundo de la programacion.

Salas de Descanso

Las salas de descanso son primordiales, aqui no todo se hace como arte de magia, asi que cuando estes frustrado con algo o no te salga todo como esperabas, puedes venir aqui a relajarte y despejar la mente.

Cafeteria

La cafeteria es un lujo para todos los alumnos, tenemos absolutamente de todo para que puedas llenarte de energias ya sea con un cafe o con una exquisita comida.
